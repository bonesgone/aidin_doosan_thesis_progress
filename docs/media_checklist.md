# Media Checklist

Use this checklist before adding media to the public repository.

## Safe To Share

- Clean photos of the robot setup.
- Short rollout videos without terminal output, file paths, or private notes visible.
- Learning curves exported from public-safe experiment summaries.
- Diagrams that explain the method without exposing implementation details.

## Avoid Sharing

- Source code screenshots.
- Raw datasets or HDF5 files.
- Checkpoints, calibration files, credentials, or private paths.
- Study notes copied from the private repository.
- Videos that show private browser tabs, terminals, or lab-only information.

## Suggested Filenames

- `assets/images/system_overview.png`
- `assets/images/task_setup.png`
- `assets/videos/scripted_demo.mp4`
- `assets/videos/policy_rollout.mp4`
- `assets/figures/training_curve_bc.png`
- `assets/figures/tactile_ablation.png`
